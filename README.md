# Project Setup

## Prerequisites
- [Node.js](https://nodejs.org/) (latest version recommended)
- [nvm](https://github.com/nvm-sh/nvm) for Node.js version management

## Setup Instructions

1. **Use the latest Node version**
    ```bash
    nvm use 22
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

3. **Start development server**
    ```bash
    npm run dev
    ```
